# Tron Wallet Portal

This folder contains frontend (index.html) and backend (server.js) for the tron wallet automation.

## To run locally

Backend:

```
node server.js
```

Frontend:

```
http-server
```

## Vercel

api/transfer.js

- created is for vercel deployment

- start vercel local dev

```
vercel dev --port 4000
```

- start local

1. node server.js
2. http-server
